codage_identity - A "no-operation" conversion function

Calling Sequence

pop_out = codage_identity(pop_in,direction,param)

Parameters

  • pop_in : the population to be converted.
  • direction : 'code' or 'decode'. This value has no influence of the state of pop_in.
  • param : a parameter list. For this function, there are no useful parameters set.
  • pop_out : a population identical to pop_in.

Description

This function is a do-nothing function. It is essentially useful to implement an evolutionnary algorithm. In an evolutionnary algorithm, we work directly on the variable and not on a binary code.
