init_func_default - A function a initialize a population

Calling Sequence

Pop_init = init_func_default(popsize,param)

Parameters

  • popsize : the number of individuals to generate.
  • param : a list of parameters. - 'dimension': the size of the vector X. - 'minbound': a vector of minimum bounds for the variable X. - 'maxbound': a vector of maximum bounds for the variable X.
  • Pop_init : a list which contains the initial population of individuals.

Description

This function generate an initial population containing pop_size individuals.
